First programming Language

In 1940, the first recognizably modern electrically powered computer was created. And the first functioning programming language designed to communicate instructions to a computer were written in the early 1950s.
In 1954, FORTRAN was invented at IBM by a team led by John Backus, it was the first widely used high level general purpose programming language to have functional implementation, as opposed to justa design on paper. When FORTRAN was invented it had a lot of bugs.
FORTRAN is a general purpose, compiled imperative programming language that is specially suited to numeric computation and scientific computation. It has been in use for over six decades in computationally intensive areas.Fortran has had numerous versions, each of which has added extensions while largely retaining compatibility with preceding versions
Fortran's design was the basis for many other programming languages. As of August 2021, Fortran was ranked 13th in the TIOBE index, a measure of the popularity of programming languages, climbing 29 positions from its ranking of 42nd in August 2020.
