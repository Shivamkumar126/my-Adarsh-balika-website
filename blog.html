About React Framework

React or React JS is a free and open source front end javascript framework for building user interface. It is maitained by Meta(formaly Facebook) and a community of individual developers and company. REact Native uses JavaScript to build cross-platform apps. React is all about components. You need to think of everything as a component. This will help you maintain the code when working a large scale projects. React is a popular and widely used at present since almost every web developer or a full-stack developer who knows HTML and wants to build more organised and dynamic websites, faster.
HTML is a language of the web, but creating entire website with HTML alone can be repetitive and hard to manage. But React uses JSX i.e it uses JavaScript with HTML tags . It takes advantage of HTML's popularity and strength as the most popular programming language, by letting you use a very similar syntax to build a interface and add dynamic features to it using Javascript. So we use React to make it easy to build and maintain a website or app. It also have some drawbacks like it uses inline templating and JSX, which might seems awkward to some developers.
